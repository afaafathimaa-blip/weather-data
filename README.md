🌦️ Weather Data Analysis Project
📌 Overview

This project analyzes weather data to understand patterns and relationships between different weather conditions such as temperature, humidity, cloud cover, wind speed, and precipitation.

📊 Features
Correlation analysis between weather variables
Visualization using heatmaps and bar charts
Distribution of different weather conditions
Basic insights into weather patterns
📁 Dataset

The dataset includes the following features:

Temperature (°C)
Feels like temperature (°C)
Humidity (%)
Cloud cover
Wind speed (mph)
Precipitation (mm)
UV index
Weather condition (text)
📈 Key Insights
Temperature and “feels like” temperature are highly correlated
Higher humidity is linked with more cloud cover
UV index tends to increase with temperature
Most days are partly cloudy, indicating stable weather
Rain and storm conditions are rare
🛠️ Tools & Technologies
Python
Pandas
